Frugal Sheets

Frugal Sheets is a spreadsheet app inspired by Google Sheets. It uses a simple Ruby on Rails backend to store sheet and user data in a Postgresql database, and a complex frontend developed in React, Javascript, utilizing a Redux data store and displayed using HTML5 and CSS3. Some of its core features are:

  • Data entry
  • Rich Text editing
  • Formula parsing
  • Data charts
